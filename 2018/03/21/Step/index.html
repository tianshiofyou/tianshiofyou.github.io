Step

hexo搭建个人专属blog

##如果在用户根目录/.ssh下，没有id_rsa.pub
git生成公钥
复制id_rsa.pub中的公钥
在github上新建

##第一步

npm install -g hexo-cli  (安装hexo脚手架)

npm install hexo-deployer-git --save (安装git部署插件)

配置

hexo init
初始化hexo

npm install
这里是自动装配需要的组件

hexo g
这里是生成

>>folder->.deploy_git->CNAME
这里部署访问域名，没有域名写github的用户名.github.io

>>folder->_config.yml
这里配置Site URL deploy的基本信息

hexo s
本地发布

写blog

hexo new post “{Name}”

部署到github

hexo g -d
Compartir
